ReactJS Minesweeper

How to Play

  1. The object of the game is to find all of the mines in the minefield without detonating them. Detonating a mine will end the game.
  2. To begin, select a difficulty level. Beginner is chosen by default.
  3. Begin clicking cells to clear unexplored territory.
  4. Numbered cells indicate the number of mines contained within surrounding cells. Use the numbers as a guide to select only squares that do not contain mines.
  5. Flag suspected mines by right clicking a cell. Flagging mines will help you remember their location so you do detonate them accidentally.
  6. Continue revealing cells. If you open all of the cells without detonating any mines you will win!
